本app是一款集社交发帖功能为一体的校园app
========
使用第三方sdk的功能：
-----
* 社交功能：集成腾讯云通信im sdk  详情https://cloud.tencent.com/product/im#sdk  10w日活量以下免费
* 对象储存功能：使用腾讯云储存，原因也是因为腾讯云的对象储存免费额度，小规模的应用基本免费.
* 短信验证功能：使用mob免费短信验证，通过审核后免费使用短信验证登录
